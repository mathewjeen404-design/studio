# **App Name**: TypeVerse

## Core Features:

- Typing Lessons: Provide a structured typing lesson system with multiple stages (warm-up, practice, test).
- Real-time Feedback: Capture keystrokes, compare with target text, and highlight correct/incorrect characters in real time.
- Performance Metrics: Calculate and display real-time metrics such as WPM, accuracy, errors, and missed keys.
- Typing Tests & Games: Offer various typing tests (1-min, 3-min, 5-min, endless) and mini-games (word challenge, accuracy challenge, speed challenge).
- Virtual Keyboard: Display a virtual keyboard that highlights typed keys and mistakes.
- Progress Tracking: Track user progress with charts for WPM, accuracy, and streaks.
- Gamification: Implement streaks, badges, and levels to gamify the typing experience.
- User Profiles: Store and display user profiles with best WPM, average accuracy, total words typed, sessions, badges, and lesson progress in Firestore.
- Leaderboard: Display a leaderboard showing top users for WPM and accuracy, fetched using efficient Firestore queries.
- Offline Mode: Enable offline mode by storing sessions locally and syncing to Firestore when online.

## Style Guidelines:

- Primary color: Deep Teal (#008080) for a professional and calming feel.
- Secondary color: Light Cyan (#E0FFFF) for highlighting interactive elements.
- Accent color: Coral (#FF7F50) to draw attention to important actions.
- Body font: 'Roboto' for readability and modern style.
- Headline font: 'Montserrat' for a bold and clear title.
- Use clear, outlined icons for navigation and features.
- Responsive grid-based layout with consistent spacing and padding.
- Subtle transitions for page changes and feedback animations for correct/wrong typing.
