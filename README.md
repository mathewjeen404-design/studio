# TypeVerse

This is a NextJS typing application created in Firebase Studio.

## Getting Started

To get started, take a look at `src/app/page.tsx`.

## Running Locally

To run this application on your local machine, please follow these steps:

1.  **Prerequisites:** Make sure you have [Node.js](https://nodejs.org/) (version 18 or later) and [npm](https://www.npmjs.com/) installed on your computer.

2.  **Download the Code:** Download the project files to a folder on your computer.

3.  **Install Dependencies:** Open a terminal or command prompt, navigate into your project folder, and run the following command to install the necessary packages:
    ```bash
    npm install
    ```

4.  **Run the Development Server:** Once the installation is complete, you can start the development server with this command:
    ```bash
    npm run dev
    ```

5.  **Open in Browser:** The app will be running at [http://localhost:9002](http://localhost:9002). Open this URL in your web browser to see your application.

That's it! Any changes you make to the code will be automatically reflected in your browser.
